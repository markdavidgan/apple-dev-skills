# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.1.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [0.1.0] - 2026-05-27

### Added
- Initial release of Apple Dev Skills
- 19 skills covering Swift 6, SwiftUI, SwiftData, iOS 26+ APIs, design, accessibility, App Store Connect, testing, and quality workflows
- 3 agents: build-agent, auditor, ios-code-reviewer
- 6 commands: /apple-check, /apple-cleanup, /apple-polish, /apple-review, /complete-feature, /setup-asc
- App Store Connect MCP server with 80+ tools
- Multi-platform support: Claude Code, Cursor, Kimi Code, Antigravity, Codex CLI, Agy
- Cross-platform installer (`install.sh`)
- Build system (`scripts/build.js`) and validation (`scripts/validate.js`)
- Kimi plugin tools: pattern-check and api-lookup
