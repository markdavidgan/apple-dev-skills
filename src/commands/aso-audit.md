Run the `asc-aso` skill on `$ARGUMENTS`.

Audits the App Store listing for discoverability (title/subtitle/keyword-field strategy, keyword repetition, localization coverage) and conversion (screenshots, preview, promotional text, PPO/custom product pages). Pulls current metadata via the App Store Connect MCP when available.
