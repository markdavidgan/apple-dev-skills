Run the `app-marketing-context` skill on `$ARGUMENTS`.

Creates or updates the persistent `app-marketing-context.md` go-to-market brief (overview, value proposition, competitors, current ASO state, goals/KPIs, resources, markets) that every other marketing/growth skill reads first. Offers to pull live listing data via the App Store Connect MCP. Distinct from `write-spec` (product requirements).
